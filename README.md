## 说明：

技术路线

Kinect (有线连接) --> 电脑 (蓝牙无线连接) --> Arduino (有线控制) --> 机械臂

Kinect将数据传送到电脑，电脑根据需要生成指令，指令通过蓝牙传到Arduino控制机械臂

重点还是电脑端是数据处理与指令生成

## 项目结构
